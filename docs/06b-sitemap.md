# Sitemap — Family Control Panel
_Version 0.1 — mai 2025 — Cycle 1 MVP_

## Kiosque (public, sans authentification)

- **Vue semaine** — écran par défaut, centré sur aujourd'hui
  - Déclarer une tâche faite (PIN enfant)
  - Valider / invalider une tâche (PIN parent)
  - Navigation jour précédent / suivant

## Admin (authentifié — login parent requis)

- **Planning**
  - Vue semaine éditable
  - Navigation entre semaines (passées en lecture seule, futures éditables)
  - Clonage de semaine
  - Validation des tâches déclarées faites (depuis la vue planning)

- **Enfants**
  - Liste des profils enfants
  - Créer / éditer / supprimer un profil

- **Catalogue de tâches**
  - Liste des tâches
  - Créer / éditer / supprimer une tâche

- **Configuration**
  - Moments de la journée (créneaux horaires matin / midi / soir)
  - Ville pour la météo du kiosque
  - Citation du moment
